# 🛒 Daily Need Store

A simple and responsive grocery store website built using **HTML** and **CSS**. This project provides users with an easy way to explore daily grocery products, view store information, and navigate through different pages such as Products, About, Contact, Payment, and Profile.

---

## 📌 Features

- 🏠 Home Page
- 🛍️ Products Page
- 👤 User Profile Page
- 💳 Payment Page
- 📞 Contact Page
- ℹ️ About Page
- 📱 Responsive and clean user interface
- 🎨 Simple HTML & CSS design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
